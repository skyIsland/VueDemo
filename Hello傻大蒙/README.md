## 运行方式

将项目拖入[HBuilderX](http://www.dcloud.io/hbuilderx.html),直接运行即可

## 使用须知

1. 页面样式主要在common/common.css
2. 数据在页面的methods->getData里获取，替换里面的链接及参数即可
3. 接口返回的字段根据已固定，若替换接口也要替换相应的字段名，页面才能正常渲染
3. 页面的分享需配置自己的appid等信息
